### 산술연산자  
```java
int a =3;
int b = 2;
a + b = 5;
a - b =1
a * b = 6;
a / b = 1.5;
++a // a+1=4가됌
System.out.print("a++") // a의 값이 프린트 되고 4로 바뀜
```

 `+`, `-`, `*`, `/` 가 있음 `%`는나머지를 구하는거임  
* 증감연산자: `++`, `--` 1을 더하거나 빼는것임  
* 이때 ``++i``는 전치라고함*  
``a = 3;  
print(++a);``   
3을 출력하고 a의 값이 4가됨
### 비교연산자  
``` java
a = 2;
b = 3;
a > b // false
b < a // true
a == b // false
a >= b// b이상
a <= b// b이하
a > b && a < b // 둘다 참일시 참
a > b && a < b// 하나라도 참이면 참
a != b // a가b랑 같지않으면
```

`>`,`<`,`==`,`>=`,`<=` 순서대로 초과, 미만, 같다, 이상, 이하

 # 반복문
## while문
```java
while(조건) {
    //실행 코드
} 
```
* 'while'문은 조건이 참일때까지 실행
    * `break;`-> 실행코드를 중지
    * `continue;`-> 이번코드 건너뛰고 다음코드 실행


## do while문
```java
do{
    // 실행 코드
}while(조건)
```
* `do whlie`은 **최소 한번**은 실행 그 후로는 while문과 같음

## for문 
```java
for(변수;조건;증감식){
    //실행 코드
}
```
* `for`문은 while문을 비교적 **간단**하게 한것임
    * for문은 중첩해서도 가능하다
# 조건문
```java
if(조건) 
{     
    //실행 코드
}
else if(조건) 
{ 
    //실행 코드
} 
else
{
    //실행 코드
}
```
* `for`문의 **조건**이 **참**일 시 코드를 실행
* `else if`-> `if'`문의 **조건**이 **거짓**인 경우 조건을 검사
* `else`->  **다 거짓**이면 최종적으로 실행(default랑 같음)
